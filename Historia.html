Historia

Big Data


Los orígenes de los grandes conjuntos de datos se remontan a las décadas de 1960 y 1970, cuando el mundo de los datos acababa de empezar con los primeros centros de datos y el desarrollo de las bases de datos relacionales.Alrededor de 2005, la gente comenzó a notar la gran cantidad de datos generados por los usuarios en plataformas como Facebook y YouTube.Ese mismo año, se desarrolló Hadoop, un marco de código abierto diseñado para almacenar y analizar grandes conjuntos de datos.Así NoSQL también ganó popularidad en esta época.El desarrollo de marcos de código abierto como Hadoop y Spark fue esencial para el crecimiento del big data, haciéndolo más accesible y económico.Con la llegada del Internet de las cosas (IoT), más objetos y dispositivos conectados generan datos sobre patrones de uso y rendimiento.
El aprendizaje automático también ha contribuido al aumento de datos.Aunque el big data ha avanzado, su utilidad sigue creciendo gracias al Cloud Computing, que ofrece escalabilidad elástica y bases de datos de gráficos para análisis rápido e integral.

Analisis de Datos


‘S’: En 1976, los investigadores de Bell Labs desarrollaron ‘S’, un conjunto de bibliotecas de macros Fortran para análisis estadístico interno. Más tarde, se reescribió en código C y se convirtió en la base de lo que hoy conocemos como R.
SQL: En 1970, se introdujo SQL, un lenguaje de consulta estructurado, que revolucionó la gestión de bases de datos y el análisis de datos.
Evolución: A lo largo de las décadas, el análisis de datos ha evolucionado desde subrutinas en Fortran hasta lenguajes como Python, R, y Julia. Estos lenguajes permiten el procesamiento y modelado estadístico, así como el manejo de grandes volúmenes de datos
.Ciencia de Datos: Hoy en día, el análisis de datos es fundamental en la ciencia de datos, la inteligencia empresarial y muchas otras disciplinas. Se ha convertido en una herramienta esencial para tomar decisiones informadas y extraer conocimiento de los datos
